## What is this?

This a simple ecommerce website template.

View the **[Site](https://toys-store-92729.web.app/)**.

**This is the client side of the website** You can fork or download this repository to customize it for your own use.

![screenshot](./src/assets/screenshot.png)

## Features

- [x] Open source (free to use, no attribution required)
- [x] Responsive design & mobile-friendly
- [x] Supports both dark and light modes
- [x] Highly customizable multi-component layout
- [x] Built with modern technologies (React, JavaScript, and DaisyUI)

## Quick Setup

1. Clone the repository:

      ```bash
      git clone https://github.com/saruar778620/unique-toy-client.git
      cd unique-toy-client
      ```

2. Ensure you have [Node.js](https://nodejs.org/) installed. Check your installation by running:

      ```bash
      node -v
      ```

3. In the project directory, install dependencies:

      ```bash
      npm install
      ```

4. Start the development server:

      ```bash
      npm run dev
      ```

5. View the site in **[http://localhost:5173/](http://localhost:5173/)**.

6. Customize the template by navigating to the `/src/components` directory.
   The page will reload if you make edits, and you will see any lint errors in the console.
